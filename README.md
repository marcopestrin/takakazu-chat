## Environment Variable
Create an `.env` file in the root:
```
# FE:
REACT_APP_PORT=
REACT_APP_BACKEND_PORT=

# BE:
BE_PORT=
BE_FRONTEND_PORT=
```
## Run local backend
```
cd be
npm run compile-ts-watch
```

and in another terminal session:

```
cd be
npm run start
```

## Run Local Frontend

```
cd be
npm run start
```
